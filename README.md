# @kared/kui

Design system - React components

## Installation

```bash
npm install kui
# or
yarn add kui
```

## Usage

```jsx
import { ThemeProvider } from "kui";

function App() {
  return <ThemeProvider>{/* Your app content */}</ThemeProvider>;
}
```
